#### func product(DataStrategyProcessor\<R>)

```cangjie
public func product<R>(p: DataStrategyProcessor<R>): CartesianProductProcessor<T, R>
```

功能：笛卡尔积组合器创建包含原始策略中元素的所有可能排列的数据策略。
对于无限策略，它首先迭代所有有限的子策略，然后才推进无限的子策略。
